# improve_acc
GPS + mmWave + Camera多模态传感器数据集，面向UAV毫米波通信波束预测研究。  
本项目针对无人机毫米波通信场景，构建了场景23（Scenario 23）的双单元传感器同步采集数据集，包含**11,387条记录** + **11,387张相机图像**，涵盖RGB图像、60GHz mmWave功率、GPS位置、速度/高度/距离/姿态等多维传感器信息。  
数据集可用于波束预测、目标追踪、主动学习、多模态融合、目标检测等多个研究方向。

# 1 项目介绍
## 1.1 关于多模态传感器数据
无人机毫米波（mmWave）通信中的波束预测需要多源传感器数据协同。单一传感器无法提供完整的空间-时间-信号特征，多模态数据融合是提升波束预测准确率的关键。

目前常见的数据方案对比：

| 方法名称 | 相关要点 |
| ------ | ------ |
| 纯GPS数据 | 只能提供位置信息，缺乏环境和信号特征 |
| 纯图像数据 | 只能提供视觉特征，缺乏精确空间坐标 |
| 纯mmWave信号 | 只能提供信号特征，缺乏空间上下文 |
| 多模态融合 | 本项目采用的方案，GPS+mmWave+Camera+运动学全覆盖 |

本项目采用**双单元传感器同步采集**方案，每个时间点同时记录7种数据模态：

| 单元 | 数据类型 | 说明 |
| ------ | ------ | ------ |
| Unit 1 | RGB相机 | BS1相机图像序列（JPEG） |
| | 60GHz mmWave | 毫米波功率测量数据（TXT） |
| | GPS | 位置坐标（TXT） |
| Unit 2 | GPS | 位置坐标（双端定位） |
| | 运动学 | 速度/高度/距离/三轴速度/俯仰/翻滚角 |

# 2 如何使用
## 2.1 数据读取
```python
import pandas as pd
df = pd.read_csv('scenario23.csv')
# 访问第一条记录的图像路径
img_path = df.iloc[0]['unit1_rgb']  # ./unit1/camera_data/image_BS1_1_16_58_42.jpg
```

## 2.2 关联图像
图像命名格式：`image_BS1_{ID}_{HH}_{MM}_{SS}.jpg`
- BS1：相机标识
- ID：记录序号
- HH:MM:SS：采集时间

## 2.3 研究方向
数据集支持以下研究方向：
- **波束预测**：利用GPS位置辅助预测最佳mmWave通信波束
- **目标追踪**：基于多传感器融合的目标追踪
- **主动学习**：减少标注数据需求的高效学习策略
- **多模态融合**：图像+mmWave+GPS+运动学数据的融合建模
- **目标检测**：配合YOLO格式bbox标注进行训练
